# 量化回测框架

## 项目介绍

本项目是一个基于Python的量化回测框架，用于测试和评估股票交易策略。框架支持从MySQL数据库获取真实股票数据，执行回测，并通过Plotly创建交互式可视化图表。

## 核心功能

1. **数据获取**：从MySQL数据库获取股票数据
2. **技术指标计算**：计算RSI、移动平均线等技术指标
3. **策略回测**：支持移动平均线交叉策略和RSI策略
4. **性能评估**：计算总收益率、年化收益率、夏普比率、最大回撤、胜率和盈亏比
5. **可视化**：使用Plotly创建交互式图表，展示收盘价曲线、买卖点和回测结果

## 依赖包

- pandas
- numpy

## 如何运行

1. 确保MySQL数据库已连接，并且包含股票数据
2. 运行CS.PY文件：

```bash
python CS.PY
```

3. 查看回测结果和可视化图表

## 可视化功能

- 弹出浏览器窗口展示交互式图表
- 滑动滑块切换不同时间区间的收盘价波动
- 红色加粗标记买点，绿色加粗标记卖点
- 鼠标悬停显示当天的收盘价、开盘价和交易额
- 右下角展示回测结果

## 数据说明

数据来源于MySQL数据库，表结构如下：

| 字段名 | 类型 | 描述 |
|-------|------|------|
| 日期 | date | 交易日期 |
| 股票代码 | varchar | 股票代码 |
| 开盘 | float | 开盘价 |
| 收盘 | float | 收盘价 |
| 最高 | float | 最高价 |
| 最低 | float | 最低价 |
| 成交额 | float | 成交额 |

## 策略说明

### RSI策略

- RSI周期: 14
- 超卖阈值: 30（低于此值买入）
- 超买阈值: 70（高于此值卖出）

### 移动平均线交叉策略

- 短期移动平均线: 5日
- 长期移动平均线: 20日
- 金叉买入，死叉卖出

## 风险控制

- 单笔交易最大亏损比例: 2%
- 最大回撤限制: 15%
- 每日VaR限制: 5%
